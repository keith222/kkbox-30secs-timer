# BOXTimer
- https://itunes.apple.com/tw/app/boxtimer/id1334123736?mt=8
- 打開盒子，讓音樂陪著一起倒數吧！
- Box Timer 除了讓你簡單設定計時器，還能在倒數時播放KKBOX精選歌單。
- 立刻開啟 Box Timer 讓時間跟著咚滋動！

## 使用工具
- [Swift](https://developer.apple.com/swift/)
- [KKBOX Open API](https://developer.kkbox.com/#/)
